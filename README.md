# Qualia

`Qualia` - это рогалик с процедурной генерацией этажей, перестрелками, комнатными энкаунтерами, магазином и боссом в конце забега.

## Сюжет

Мир `Qualia` - это искаженное внутреннее пространство игрока, собранное из страха и памяти. Герой спускается все глубже через слои подземелья дабы достичь своего внутреннего "я", вырваться из цикла и наконец проснуться.

## Что есть в игре

- 5 этажей с разной структурой прохождения
- процедурная генерация уровней
- обычные враги с разными паттернами движения и атаки
- лабиринтный этаж
- магазин с улучшениями за валюту `жар`
- финальный босс с несколькими фазами
- миникарта
- меню паузы, настройки и debug HUD

## Управление

- `W`, `A`, `S`, `D` - движение
- `ЛКМ` - стрельба
- `Left Shift` / `Right Shift` - уворот
- `Space` - взаимодействие
- `P` - пауза
- `Enter` - начать игру из главного меню
- `F1` - открыть / скрыть debug HUD
- `Esc` - выйти из игры

Для меню и магазина используется мышь.

## Запуск

1. Установить зависимости:

```bash
pip install pygame-ce
pip install -r requirements.txt
```

2. Запустить игру из корня проекта:

```bash
python qualia/game.py
```

## Основные алгоритмы

- `BSP dungeon generation` - основа генерации обычных этажей: карта рекурсивно делится на области, внутри которых создаются комнаты и соединяются коридорами.
- `DFS maze generation` - используется для лабиринтного этажа: генератор строит связный лабиринт через обход в глубину с возвратом назад по стеку.
- `Flood fill / BFS validation` - после генерации уровень проверяется на достижимость ключевых зон и корректность проходов.
- `Game loop + delta time` - обновление движения, стрельбы, таймеров и анимаций идет с учетом времени кадра, а не количества FPS.
- `AABB collision` - столкновения игрока, врагов, пуль, стен и подбираемых предметов считаются через пересечение хитбоксов.
- `Line of sight` - враги обнаруживают игрока только при наличии прямой видимости, а не знают его позицию заранее.
- `Phase-based boss FSM` - поведение босса меняется по фазам в зависимости от количества здоровья.
- `Ricochet projectiles` - отдельные типы вражеских пуль умеют отскакивать от стен с постепенной потерей скорости.
- `Fog of war minimap` - миникарта открывается постепенно по мере исследования уровня, а не показывает весь этаж сразу.
